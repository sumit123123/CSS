Implementation of CSS

CSS Can be implement in your html by 4 methods
1. Create a css file and link it with your html page Code like These :- rel="stylesheet" href="style.css"
2. Using the html style tag in your html body Code like these style styl Inside these style tag you can write css Property
3. Inline css You can give the css property on the element itself
4. CSS By JavaScript In JavaScript you have some code like these document.body.style = CSS Property
